# effective_mobile-test
Тестовое задание на позицию Junior системный аналитик

## О проекте
Тестовое задание для позиции Junior системный аналитик в компанию [Effective Mobile].
Выполнено в формате, приближенном к реальным рабочим артефактам: анализ требований, проектирование API, архитектурная схема.

## О репозитории
Ответы на вопросы тестового задания

## Структура репозитория
- docs/ - ответы на вопросы тестового задания и текстовые артефакты
- diagram/ - BPMN-диаграммы и схемы
- image/ - изображения
- README.md - путеводитель по решению
  
## Технологии и форматы
- Анализ требований — Markdown, таблицы
- API — REST, JSON
- Архитектура — Mermaid (flowchart)

## Как читать решение
### Задание 1
1. Сначала откройте файл [`docs/task1.md`](docs/task1.md) - в нём описание задания и что нужно сделать
2. После этого откройте [`docs/task1_fullAnswers.md`](docs/task1_fullAnswer.md) - подробные ответы на вопросы тестового задания

### Задание 2
1. Сначала откройте файл [`docs/task2.md`](docs/task2.md) - в нём описание задания и что нужно сделать
2. После этого откройте [`docs/task2_fullAnswer.md`](docs/task2_fullAnswer.md) - подробные ответы на вопросы тестового задания

### Задание 3
1. Сначала откройте файл [`docs/task3.md`](docs/task3.md) - в нём описание задания и что нужно сделать
2. Затем посмотрите [`diagrams/architecture_test.mermaid`](diagrams/architecture_test.mermaid) - верхнеуровневая архитектурная блок-схема
